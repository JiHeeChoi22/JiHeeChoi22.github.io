- [ ] 다양한 화면 크기(소형/중형/대형 기기)에서 UI 확인
- [ ] 가로/세로 모드 전환 시 레이아웃 정상 동작
- [ ] 시스템 폰트 크기 변경에 따른 텍스트 표시

### 실행 성능
- [ ] Cold Start 시간 (10초 이내 권장)

- [ ] GPS/블루투스 사용 시 배터리 소모량
